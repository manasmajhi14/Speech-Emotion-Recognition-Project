# Speech-Emotion-Recognition-Project
This data science project uses librosa to perform Speech Emotion Recognition. SER is the process of trying to recognize human emotion and affective states from speech.

# Datasets used in this project:
1. Crowd-sourced Emotional Mutimodal Actors Dataset (Crema-D)   Link: "https://drive.google.com/drive/folders/1fo99yJXrrPjcNHqqs3uajWll5uFOObvU?usp=drive_link"
2. Ryerson Audio-Visual Database of Emotional Speech and Song (Ravdess) Link: "https://drive.google.com/drive/folders/1Qvdk3Qzmh5z5ZATGdTW8p66M6MBIBrP7?usp=drive_link"
3. Toronto emotional speech set (Tess) Link: "https://drive.google.com/drive/folders/1zIOb4paExm9_syuzWs4MtSSoszbM0Vdw?usp=drive_link"
